# SCU-Course
## 使用方法
在`setting.py` 文件中配置学号与密码  
运行`python main.py`  
若要指定文件保存路径 运行`python main.py -p [指定路径]` 即可  
若要开启手动识别验证码（不建议）,运动`python main.py -m `即可  
## requirement
* BeautifulSoup
* pandas
* requests
* dddocr  
请使用 `pip install` 手动下载上面的包
